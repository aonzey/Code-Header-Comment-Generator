# 代码头注释生成器 (Code Header Comment Generator)

## 项目描述

一个简单易用的代码头注释生成工具，帮助快速生成标准化的代码文件头部注释。支持多种编程语言风格，提供可视化配置界面。

直达:https://aonzey.github.io/Code-Header-Comment-Generator/

## 功能特性

- **多语言支持**：支持 C/C++/Java/JS/Go/Rust、Python、HTML/XML、Shell/Bash、Lua 等多种语言风格
- **可视化配置**：直观的界面配置注释字段
- **拖拽排序**：使用 Sortable.js 实现字段拖拽排序功能
- **实时预览**：右侧实时预览生成的注释效果
- **一键复制**：支持一键复制生成的注释到剪贴板
- **本地缓存**：自动保存配置到 localStorage，下次打开自动恢复
- **日期快捷输入**：日期类型字段支持双击快速填入当前时间
- **响应式设计**：适配不同屏幕尺寸

## 使用方法

1. **选择语言风格**：在左侧面板选择目标编程语言的注释风格
2. **配置字段**：
   - 添加字段：点击"+ 添加新条目"按钮
   - 删除字段：点击字段右侧的"×"按钮
   - 拖拽排序：拖动字段左侧区域调整顺序
   - 快捷日期：双击日期字段的值输入框快速填入当前时间
3. **预览生成结果**：右侧面板实时显示生成的注释
4. **复制使用**：点击"复制代码"按钮将注释复制到剪贴板
5. **清除缓存**：点击"🗑️ 清除缓存"按钮恢复默认设置

## 支持的注释风格

| 语言风格 | 注释符号 | 示例 |
|---------|---------|------|
| C-style | `/* ... */` | C, C++, Java, JavaScript, Go, Rust |
| Python | `# ...` | Python, Ruby, YAML |
| HTML/XML | `<!-- ... -->` | HTML, XML, Vue模板 |
| Shell | `# ...` | Shell, Bash, Zsh |
| Lua | `--[[ ... ]]` | Lua |

## 使用提示

1. 字段的"条目名称"如果留空，生成注释时不会显示冒号和名称，只显示值内容
2. 包含"date"或"日期"关键词的字段会自动识别为日期字段，支持双击快捷输入
3. 配置会自动保存到浏览器本地存储，刷新页面或关闭浏览器后重新打开会自动恢复
4. 至少保留一个字段，不能全部删除
